HTML

软件结构

C/s : client / server 客户端/服务器

​ 服务器发生改变，客户端必须改

​ Eg : QQ

​ 最大特点 : 安全

b/s : Browser / server 浏览器 / 服务器

​ 服务器发生改变，客户端不需要改

​ 最大特点 : 方便 , 简单

B/s 资源分类

​ 静态资源 : 固定的 , 谁访问都一样

​ 使用技术 : html , css , js

​ 动态资源 : 不固定的 , 数据从数据库来 , 谁访问都可能不一样

​ Eg : 登录成功后 , 欢迎 xxx 的登录

​ 使用技术 : jsp servlet asp php

静态资源 :

​ Html : 页面主体 , 页面框架

​ Css : 美化

​ Js : 可以使页面有动态效果

Html : 超文本标记语言

超文本与纯文本 纯文本 : 只能保存文字 , 不能存图片 , 视频 , 不能传输样式 eg .txt 超文本 : 包含特殊标记的文本文件 eg : html .doc .md
标记语言 由标签组成的语言 : <标签内容> ​ 问题 : 1.XHTML HTML的关系 ​ 2.什么是 xml

Html : .html .htm

​ 网站首页 : index 或 default

Html 的特点 :

​ 1.标签分类 : 单标签 <br />

​ 双标签<a> </a> <A> </A>

​ 2.不区分大小写 , 建议全部小写
​ 3.标签可以嵌套 , 不可以交叉

Html文件 :

​ 标签 :

​ <img src=”图片路径”/>

​ 相对路径与绝对路径

​ 相对路径 : 以./开始 , 默认可以省略

​ 绝对路径 : 以协议开始 [ files ://…http://… ftp://]

​ 列表 : ul : 无序 ol : 有序

​ Java集合 collection map

​ Collection : list 有序 底层是数组

​ set 无序 底层是链表
